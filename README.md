# 📊 Sales Data Analysis Dashboard (Excel)

## 🔹 Overview

This project is an interactive sales dashboard built using Microsoft Excel.

## 🔹 Features

* Pivot Tables for data analysis
* Bar Chart (Sales by Region)
* Line Chart (Monthly Sales Trend)
* Slicers for dynamic filtering (Region, Product)
* KPI Cards (Total Sales, Quantity, Top Region)

## 🔹 Insights

* West region has highest sales
* March recorded peak sales
* Laptop category contributes highest revenue

## 🔹 Tools Used

* Microsoft Excel
